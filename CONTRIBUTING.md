# 기여 가이드

dotori.md에 기여해 주셔서 감사합니다!

## PR 정책

**모든 Pull Request는 100% 수용합니다.** 별도의 리뷰 과정 없이 머지됩니다.

## 기여 방법

1. 이 저장소를 **Fork** 합니다.
2. 정부 공식 사이트에서 HWP 공문서를 다운로드합니다.
3. **[hwp2md](https://github.com/leaf-kit/hwp2.md)**로 변환합니다:
   ```bash
   hwp2md 공고문.hwp -k
   ```
4. 원본 `.hwp`와 변환된 `.md`를 해당 부처 폴더에 넣습니다.
5. 커밋하고 PR을 보냅니다.

## 파일 이름 규칙

```
{연도}_{문서명}.hwp
{연도}_{문서명}.md
```

- 공백 대신 밑줄(`_`) 사용
- 원본과 변환 파일은 반드시 쌍으로 제출

## 커밋 메시지

```
docs: {부처명} {문서 설명} 추가
```

## 새 부처 폴더 추가

기존 폴더 목록에 없는 부처의 문서도 환영합니다. `docs/부처명/분야/` 구조로 폴더를 만들어 주세요.
