# dotori.md

**"다람쥐가 도토리를 모으듯, 공문서를 차곡차곡 마크다운으로."**

[![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![GitHub Stars](https://img.shields.io/github/stars/leaf-kit/dotori.md?style=social)](https://github.com/leaf-kit/dotori.md)
[![PRs Welcome](https://img.shields.io/badge/PRs-100%25_welcome-brightgreen.svg)](https://github.com/leaf-kit/dotori.md/pulls)

**dotori.md**는 접근하기 어려운 HWP 형식의 대한민국 공문서들을 누구나 읽기 쉽고 활용하기 좋은 **Markdown(.md)** 형식으로 변환하여 큐레이션하는 오픈소스 저장소입니다.

폐쇄적인 포맷에 갇혀 있던 공공의 정보를 자유로운 광장으로 끌어내어, 더 많은 사람이 정보를 **검색하고, 읽고, 공유**할 수 있도록 돕습니다.

---

## 왜 '도토리'인가요? 🤔

공문서는 우리 사회의 소중한 기록이자 자산입니다. 하지만 전용 뷰어가 있어야만 볼 수 있는 HWP 포맷은 정보의 자유로운 흐름을 방해하곤 합니다.

| | 단계 | 설명 |
|---|------|------|
| 🌰 | **수집** | 다람쥐가 겨울을 대비해 도토리를 모으듯, 중요한 공문서 데이터를 정성껏 모읍니다. |
| ⚙️ | **변환** | 고성능 변환 도구인 **[hwp2md](https://github.com/leaf-kit/hwp2.md)**를 사용하여 표, 이미지, 복잡한 행정 번호 체계를 깔끔한 마크다운으로 탈바꿈합니다. |
| 🌍 | **공유** | 마크다운으로 변환된 문서는 GitHub에서 바로 읽을 수 있고, 검색 엔진에 잘 노출되며, 어디서든 수정과 배포가 자유롭습니다. |

---

## 누구나 참여할 수 있습니다 🙌

**dotori.md는 모든 Pull Request를 100% 수용합니다.**

정부24, 고용노동부, 행정안전부, 교육부 등 어떤 정부 기관의 공문서든 상관없습니다. HWP 파일을 마크다운으로 변환해서 PR을 보내주시면, 검토 없이 곧바로 머지합니다.

> 공문서 하나를 변환하면, 그 문서가 필요한 모든 사람이 혜택을 받습니다.
> 여러분의 작은 기여가 모이면, 대한민국 공문서 전체가 열리는 날이 옵니다.

## 변환 방법 🔧

**[hwp2md](https://github.com/leaf-kit/hwp2.md)** 도구를 사용하면 HWP 파일을 깔끔한 마크다운으로 변환할 수 있습니다.

### 1. hwp2md 설치

```bash
# Homebrew
brew tap leaf-kit/hwp2.md && brew install hwp2md

# 또는 소스에서 빌드
git clone https://github.com/leaf-kit/hwp2.md.git
cd hwp2.md && cargo build --release
```

### 2. 변환

```bash
# 기본 변환
hwp2md 공고문.hwp

# 공문서 서식(-k) 적용 권장
hwp2md 공고문.hwp -k
```

### 3. PR 보내기

```bash
# 이 저장소를 fork한 뒤
git clone https://github.com/<your-id>/dotori.md.git
cd dotori.md

# 원본 HWP와 변환된 MD를 해당 부처 폴더에 복사
cp 공고문.hwp docs/고용노동부/장려금/
cp 공고문.md  docs/고용노동부/장려금/

# 커밋 & PR
git add .
git commit -m "docs: 고용노동부 고용창출장려금 공고문 추가"
git push origin main
# → GitHub에서 Pull Request 생성
```

## 폴더 구조 📂

공문서는 **부처명 → 분야** 2단계 구조로 정리합니다. 각 폴더에는 원본 `.hwp`와 변환된 `.md`가 함께 저장됩니다.

```
docs/
├── 고용노동부/
│   ├── 채용공고/          # 채용시험, 경력경쟁채용 등
│   │   ├── 2025_경력채용_공고문.hwp
│   │   └── 2025_경력채용_공고문.md
│   ├── 장려금/            # 고용창출, 청년일자리, 장애인고용 등
│   └── 훈련사업/          # K-디지털, 직업훈련 등
│
├── 행정안전부/
│   ├── 서식/              # 공공서식, 민원양식 등
│   └── 지침/              # 업무지침, 운영규정 등
│
├── 기획재정부/
│   ├── 예산/              # 예산안, 기금운용계획 등
│   └── 세제/              # 조세지출, 세법개정 등
│
├── 교육부/
│   ├── 입시/              # 대입, 수능, 학점제 등
│   └── 정책/              # 교육과정, 학교정책 등
│
├── 인사혁신처/
│   ├── 채용/              # 공무원 채용공고, 시험계획 등
│   └── 보수/              # 공무원보수, 수당 지침 등
│
├── 국토교통부/
│   ├── 건축/              # 건축인허가, 주택정책 등
│   └── 교통/              # 도로, 철도, 항공 등
│
├── 보건복지부/
│   ├── 복지/              # 기초생활, 아동, 노인 복지 등
│   └── 의료/              # 건강보험, 의료정책 등
│
├── 과학기술정보통신부/
│   ├── 디지털/            # 디지털전환, SW정책 등
│   └── 연구/              # R&D, 연구과제 공고 등
│
├── 법무부/
│   ├── 법령/              # 법률, 시행령, 시행규칙 등
│   └── 인권/              # 인권정책, 법률구조 등
│
├── 환경부/
│   ├── 환경/              # 환경영향평가, 폐기물 등
│   └── 기후/              # 탄소중립, 기후대응 등
│
└── 기타/                  # 위 분류에 해당하지 않는 문서
```

> 위 구조에 없는 부처(외교부, 국방부, 문화체육관광부 등)의 문서도 환영합니다.
> 새로운 부처 폴더를 직접 만들어서 PR을 보내주시면 됩니다.

## 컨벤션 📏

### 파일 이름 규칙

```
{연도}_{문서명}.hwp
{연도}_{문서명}.md
```

| 규칙 | 설명 | 예시 |
|------|------|------|
| 연도 접두사 | 문서 발행 연도 4자리 | `2025_` |
| 문서명 | 공백 대신 밑줄(`_`), 한글 사용 가능 | `고용창출장려금_공고문` |
| 쌍으로 저장 | 원본 `.hwp`와 변환 `.md`를 반드시 함께 | `2025_공고문.hwp` + `2025_공고문.md` |
| 소문자 확장자 | `.hwp`, `.hwpx`, `.md` | |

**예시:**

```
docs/고용노동부/장려금/
├── 2025_고용창출장려금_공고문.hwp
├── 2025_고용창출장려금_공고문.md
├── 2025_청년일자리도약장려금_공고문.hwp
└── 2025_청년일자리도약장려금_공고문.md
```

### 커밋 메시지 규칙

```
docs: {부처명} {문서 설명} 추가
```

**예시:**

```
docs: 고용노동부 2025 고용창출장려금 공고문 추가
docs: 행정안전부 공공서식한글 보급계획 추가
docs: 교육부 2025 수능 시행계획 추가
```

### 폴더 추가 규칙

- 1단계: **부처명** (정식 명칭, 한글)
- 2단계: **분야** (짧고 직관적인 키워드)
- 3단계 이하는 필요 시 자유롭게 생성 가능

```
docs/새부처명/
└── 분야명/
    └── (선택) 세부분야명/
```
